#### 定义函数

```python
def my_abs(x):
    if x >= 0:
        return x
    else:
        return -x
```

#### 默认参数

函数的默认参数的地址是不会改变的

**默认参数必须指向不变对象**

```python
def power(x, n=2):
    pass
```

#### 可变参数

可变参数在函数内部是一个`tuple`元组

```python
def calc(*number):
    pass

# 把 list 当可变参数传入方法
num = [1,2,3]
calc(*num)
```
